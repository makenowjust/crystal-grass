# crystal-grass

[Grass] interpreter written in Crystal.

[Grass]: http://www.blue.sky.or.jp/grass/

## Installation

```console
$ git clone https://github.com/MakeNowJust/crystal-grass
$ crystal build --release -o grass src/grass.cr
```

## Usage

```console
$ ./grass hello.www
```
